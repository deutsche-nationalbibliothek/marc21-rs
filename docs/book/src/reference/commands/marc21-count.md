# marc21-count(1)

## NAME

*marc21-count* --- Print the number of records in the indput data.

## SYNOPSIS

`marc21 count` [_options_] [_path_]...\
`marc21 cnt` [_options_] [_path_]...

## DESCRIPTION

The `count` is used to determine the number of records contained in
the input.
